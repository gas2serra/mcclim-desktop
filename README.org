* McClim Desktop 

** Usage

*** CL

#+BEGIN_SRC lisp

(desktop:launch-application 
         (desktop:find-application "climacs"))

(desktop:launch-application 
	  (desktop:find-application "climacs") 
	  :args '("/etc/passwd"))
#+END_SRC

or 

#+BEGIN_SRC lisp

(desk:launch-app "ernestine") 

(desk:launch-app :climacs :args '("/etc/passwd"))

#+END_SRC

*** GUI

#+BEGIN_SRC lisp
(desk:launch-app "launcher") 
#+END_SRC

** Installation

Clone the project into local-projects of quicklisp.

#+BEGIN_SRC lisp
(ql:quickload :mcclim-desktop)
#+END_SRC
